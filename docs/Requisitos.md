# 📋 Requisitos do Projeto — BSbVibes

## 🛠️ Requisitos Funcionais (No-Code / Wix)
* **RF-001:** O sistema deve apresentar de forma visual os principais serviços de um SóciosRB (Museu, Ingressos, Visitas e etc).
* **RF-002:** O sistema deve exibir detalhes estruturais de acessibilidade física (rampas, elevadores e rotas adaptadas).
* **RF-003:** O sistema deve conter um menu de navegação superior com links de âncora que realizam o scroll automático para as respectivas seções da página.

## ⚙️ Requisitos Não-Funcionais / Técnicos (Vibecode)
* **RNF-001:** O sistema deve integrar um componente externo em HTML/CSS customizado (via Vibecode) no rodapé da página para centralizar dados de utilidade pública e rotas integradas.
* **RNF-002:** A interface deve ser desenvolvida sob o paradigma No-Code com tempo de resposta imediato no carregamento dos elementos visuais e interativos em navegadores web convencionais.
* **RNF-003:** O componente de código embutido no rodapé deve utilizar codificação HTML/CSS limpa e tradicional para garantir total compatibilidade com as restrições de segurança e execução dos servidores da plataforma de hospedagem.
